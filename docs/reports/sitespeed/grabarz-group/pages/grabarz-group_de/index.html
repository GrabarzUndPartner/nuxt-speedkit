Avoid extra requests by setting cache headers (cacheHeaders)The page has 3 requests that are missing a cache time. Configure a cache time so the browser doesn't need to download them every time. It will save 30.9 kB the next access.70
Description: The easiest way to make your page fast is to avoid doing requests to the server.
