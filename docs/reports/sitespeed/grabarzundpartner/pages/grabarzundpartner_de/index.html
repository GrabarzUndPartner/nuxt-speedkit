Avoid CPU Long Tasks (longTasks)The page has 1 CPU long task with the total of 115 ms. Description: Long CPU tasks locks the thread. Avoid extra requests by setting cache headers (cacheHeaders)The page has 1 request that are missing a cache time. Configure a cache time so the browser doesn't need to download them every time. It will save 5.2 kB the next access.90
Description: The easiest way to make your page fast is to avoid doing requests to the server. Setting a cache header on your server response will tell the browser that it doesn't need to download the asset again during the configured cache time! Always try to set a cache time if the content doesn't change for every request.
